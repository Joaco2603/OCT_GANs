## DESPUES DE TENER EL DATASET

### Para GENERAR imagenes (RAPIDO, funciona en CPU):

```powershell
python progan_local.py --mode generate --num_images 10
```

Esto usara los pesos pre-entrenados (si los tienes) para generar
imagenes nuevas en segundos.

### Para ENTRENAR (LENTO en CPU):

```powershell
python progan_local.py --mode train
```

‚ö†Ô∏è  ADVERTENCIA: En CPU, entrenar el modelo completo puede tomar
   DIAS o SEMANAS. Recomendaciones:

   1. Usa Google Colab (GRATIS con GPU):
      - Abre: ProGAN/progan_oct_main.ipynb
      - Subelo a Google Colab
      - Ejecuta alli (tiene GPU Tesla T4 gratis)
   
   2. Prueba local con dataset peque√±o:
      python progan_local.py --mode train --max_images 100
      (Solo para probar que funciona)
   
   3. Consigue GPU NVIDIA local si planeas entrenar mucho
